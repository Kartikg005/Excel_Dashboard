# 🚗 Car Sales Analysis Dashboard

An interactive **Microsoft Excel Dashboard** built using the **Cars Dataset from Kaggle**. This project demonstrates data cleaning, data analysis, and dashboard design using Pivot Tables, Pivot Charts, KPI Cards, and Slicers.

---

## 📌 Project Overview

The Car Sales Analysis Dashboard provides an interactive way to analyze vehicle data. It transforms raw data into meaningful insights by visualizing key metrics such as brand distribution, average prices, fuel types, transmission types, manufacturing years, and engine sizes.

The dashboard enables users to explore the dataset dynamically using slicers and helps identify trends for better decision-making.

---

## 🎯 Objectives

- Clean and prepare the dataset for analysis.
- Build Pivot Tables for data summarization.
- Create KPI Cards to display key metrics.
- Design Pivot Charts for data visualization.
- Add interactive slicers for filtering.
- Develop a professional Excel dashboard.
- Generate business insights from the data.

# 📋 Workflow

### Step 1 – Dataset Collection

- Download the Cars Dataset from Kaggle.
- Save the dataset as an Excel Workbook (.xlsx).

---

### Step 2 – Data Import

- Import the dataset into Microsoft Excel.
- Convert the dataset into an Excel Table (Ctrl + T).

---

### Step 3 – Data Cleaning

- Remove duplicate records.
- Check for missing values.
- Verify data types.
- Organize the dataset for analysis.
